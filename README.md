# 📜 Cordel Moderno

Projeto desenvolvido com HTML5 e CSS3, inspirado no poema Cordel Moderno, de Milton Duarte.

A proposta do site é unir literatura e tecnologia, apresentando o cordel em um formato visual moderno, com uso de imagens de fundo, tipografia estilizada e efeito parallax.

### 🎯 Objetivo

Praticar:

Estruturação semântica com HTML

Estilização avançada com CSS

Uso de fontes externas (Google Fonts)

Efeito de background fixo (parallax)

Organização visual de textos

---

### 🚀 Tecnologias utilizadas

HTML5

CSS3

Google Fonts

### ✨ Funcionalidades

Layout simples e elegante

Texto em formato de poesia (cordel)

Seções com imagens de fundo fixas

Destaque visual para trechos do poema

Responsividade com uso de vw

---

### 💡 Aprendizados

Nesse projeto eu aprendi e pratiquei:

Uso de variáveis CSS (:root)

Aplicação de background com efeito parallax (background-attachment: fixed)

Trabalhar com tipografia personalizada

Melhor organização de layout com seções

Estilização de textos poéticos
